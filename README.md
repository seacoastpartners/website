# Website

> Company website

## Development

Install dependencies:

```zsh
npm install
```

Run with hot-reload:

```zsh
npm run dev
```

Compile and minify for production:

```zsh
npm run build
```

Preview production build:

```zsh
npm run preview
```

Lint and fix files:

```zsh
npm run lint
```
